# CES Summary

Processes CES interview transcripts into a brief monthly summary. Publishes to Confluence and posts a condensed version to Slack.

## How it works
1. Reads up to 4 transcripts (one per session)
2. Codes key moments against this month's core questions
3. Pulls standout quotes and flags unexpected findings
4. Generates a cross-session summary with 3-4 key findings, patterns, and recommendations
5. Publishes to the monthly CES Confluence page (replaces placeholder)
6. Posts condensed version to Slack

## Trigger phrases
`ces summary`, `summarise ces`, `ces write-up`, `ces transcripts`, `write up ces`

## Outputs
- Summary content on the monthly Confluence page
- Slack post to #onboarding-squad-public
