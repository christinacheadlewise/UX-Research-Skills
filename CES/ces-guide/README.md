# CES Guide

Topline discussion guide for CES Rolling Research. Fixed intro (2 min) and conclusion (2 min) stay the same every month; the core section (26 min) rotates based on squad input.

## How it works
1. Checks for an existing guide or creates from template
2. Pulls this month's focus from Slack thread or direct input
3. Generates 3-5 core questions with probes
4. Saves locally and publishes to Confluence under the monthly CES page

## Trigger phrases
`ces guide`, `update ces guide`, `tweak the ces questions`, `ces middle questions`

## Outputs
- `CES/[Month Year]/ces-discussion-guide.md`
- Confluence child page: "Discussion Guide: CES Rolling Research — [Month Year]"
