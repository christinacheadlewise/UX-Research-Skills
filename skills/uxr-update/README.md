# UXR Update

Weekly Monday Confluence refresh. Walks through what's changed and updates the hub, Active Studies, and Research Repository pages in sync.

## How it works
1. Reads current state of all managed pages
2. Walks Christina through changes: new studies, status updates, completed work
3. Updates In Progress / Recently Completed tables across all pages
4. Schedules a Slack reminder for the following week (shifts to Tuesday if Monday is a UK bank holiday)

## Trigger phrases
`uxr update`, `weekly update`, `update confluence`, `monday update`

## Outputs
- Updated Confluence pages (hub, Active Studies, Research Repository)
- Scheduled Slack reminder for next week
