# UXR Slack

Takes a research slide deck and generates a formatted Slack post summarising the findings. Can post directly to a channel or output as text.

## How it works
1. Reads the deck (PDF, file path, or pasted content)
2. Extracts background, methodology, findings, recommendations, and quotes
3. Asks about target channel, whether it's part of a series, and lead insight
4. Generates a formatted post: emoji header, link, background, methodology, TL;DR bullets, key insights with quotes, and recommendations
5. Iterates until approved, then posts or outputs text

## Trigger phrases
`uxr slack`, `write a slack summary`, `slack post`, `share this research`, `write up for slack`, `summarise this deck for slack`

## Outputs
- Formatted Slack post (posted directly or as copyable text)
