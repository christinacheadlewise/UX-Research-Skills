# UXR Sessions

Pushes live session information into the "This Week" panel on the Confluence hub page. Can pull from Slack or direct input.

## How it works
1. Asks what sessions are happening this week (or reads from Slack)
2. Formats session details: study name, times, viewing links
3. Updates the "This Week" panel on the hub page (follows wipe guard)
4. Optionally cross-posts to a Slack channel

## Trigger phrases
`uxr sessions`, `push sessions`, `update this week`, `sync sessions from slack`, `add live sessions`

## Outputs
- Updated "This Week" panel on Confluence hub page (4125630194)
- Optional Slack cross-post
